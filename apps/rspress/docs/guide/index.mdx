# HI

欢迎来到我的`blog`，我是一个`前端仔`，这里是用来记录、学习、分享的 😏。

目前点亮`工程化`,`rust`,`vite`,`react`,`vue`,`sql`，`electron`，`webpack`等一些技能树。（爱你 ~~😘）；

前端技术
- 基于`rspress`进行静态站点生成的
- 此项目使用`wpa`实现 web 离线缓存应用
- 使用`turbo`,合并`blog`和配套的`tauri`图床应用项目
- `turbo`打开项目和打包对于CI/CD的构筑是真滴快😏
- 使用`pnpm`的`monorepo`进行包管理
- 引入`ant-design`,`tailwind`,`styled-components`,
- 动画小人是使用`oh-my-live2d`配合书写项目的插件进行引用的


服务端
- 使用`docker`+`jenkins`+`nginx`集成`CI\CD`工作流
- 利用`github`的`webhook`实现绝赞的自动更新
